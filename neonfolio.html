The Vision

Neonfolio is a developer-first portfolio framework designed to get out of the way. It prioritizes performance, SEO, and ease of content authoring, allowing developers to treat their portfolio as code.

The Philosophy

Most portfolio templates are bloated. Neonfolio is stripped back to the essentials. It uses a file-system based routing for case studies (MDX) and a zero-runtime CSS engine to ensure perfect Lighthouse scores.

Technical Deep Dive

Built on the Next.js App Router, Neonfolio leverages React Server Components to minimize client-side JavaScript.

  • MDX Engine Write case studies in Markdown with embedded React components.
  • Optimization Automatic format conversion and lazy-loading for all assets.
  • Theming API Robust dark/light mode implementation using CSS variables and system preference.
