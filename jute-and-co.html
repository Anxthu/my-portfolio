Overview

This project is a high-end, aesthetic e-commerce web application designed for a sustainable fashion brand. It focuses on delivering a premium user experience through smooth animations, interactive 3D elements, and a clean, minimalist design. Built as a Single Page Application (SPA), it ensures seamless navigation and instant feedback.

Key Features

  • Immersive 3D Carousel Scroll-driven 3D product carousel using Framer Motion's useScroll hook, creating a "scrollytelling" experience.
  • Dynamic Product Filtering Custom-built dropdowns and real-time state management for sorting and filtering products without page reloads.
  • Seamless Shopping Cart Global CartContext calculates totals dynamically, handles duplicate items intelligently, and provides instant UI updates.

Architecture & Data Flow

The application follows a component-based architecture with separate concerns for logic and UI.

  • State Management React Context API avoids prop drilling, ensuring cart state is accessible across the Navbar, Product Pages, and Checkout.
  • Component Structure Atomic Design principles used for reusable UI blocks like atomic ProductCards and Navbars.
  • Data Handling Static data simulation via local JS files creates an instant-loading prototype ready for backend integration.

Future Roadmap

While currently a polished prototype, the roadmap includes Firebase Auth integration, migration to a MongoDB/Firestore backend, and Stripe payment processing.
